.. _require:

Requirements
============

Web server
----------

Since the floorplan's interface is based entirely in your browser, you'll need a
web server (such as Apache, nginx, :term:`IIS`) to install this repo's files into.

PHP
---

* You need PHP 7.0.33 or newer, with ``session`` support, the Standard PHP Library
  (SPL) extension, JSON support, and the ``mbstring`` extension (see :term:`mbstring`).


Database
--------

supports MySQL-compatible databases.

* MySQL 5.5 or newer
* MariaDB 5.5 or newer

.. seealso:: :ref:`faq1_17`

Web browser
-----------

To access the floorplan you need a web browser with cookies and JavaScript enabled.
